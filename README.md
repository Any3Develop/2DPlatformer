# Тестовое задание — 2D прототип с «гравитацией к центру»

Небольшой прототип, сделанный за ~3 часа без спешки и без попыток что-то «вылизывать».  
Минималистичная реализация базовой механики для тестового задания.

## Описание
Персонаж перемещается по сторонам объекта, как будто гравитация тянет его к центру формы.  
Изначально требовался один прямоугольник, но я **не удержался и добавил поддержку других фигур** — система работает с произвольными формами.

Поддержано:
- Ходьба по любой стороне фигуры  
- Прыжок от текущей поверхности  
- Плавный переход по углам/стыкам граней  
- Невозможно «упасть» за пределы формы  
- Поддержка нескольких фигур на сцене  
- **Можно переходить между фигурами — достаточно коснуться другой формы, и персонаж «перепривяжется» к её поверхности**

## Управление
### ПК
- **A / D** — движение  
- **Space** — прыжок  

### Мобильные устройства
- Экранные кнопки: два **Move**, одна **Jump**

## Статус
Это **тестовое задание**, выполненное в полном объёме для демонстрации подхода.
